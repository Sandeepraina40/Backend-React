# Social Media App

React + Redux Toolkit + React Router app that fetches posts from `https://jsonplaceholder.typicode.com/posts`, shows a grid on `/` and a detail view on `/item/:id`.

## Run

```bash
npm install
npm run dev
```

Then open the printed local URL.

## Build

```bash
npm run build && npm run preview
```

## Notes
- Uses Redux Toolkit (thunk included) for async fetching and state.
- Placeholder images via `https://picsum.photos/200?random={id}`.
- Responsive dark theme styled to match the provided design.
